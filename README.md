Monterrey Theme
---------------

Monterrey is a theme for pelican inspired by Attila.

### Setup

	git clone https://github.com/feeliks-ahumada/Monterrey.git
	pelican-themes --symlink ~/path/to/repo/Monterrey --verbose
	pelican-themes -l
